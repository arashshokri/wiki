

# Extension API

برای ارسال تماس به سمت یک داخلی استفاده می‌شود. عملکرد آن مشابه Extension است، با این تفاوت که از طریق وب‌‌سرویس، شماره داخلی مقصد را دریافت و تنها تماس را برای آن داخلی ارسال می‌‌نماید (برای توضیحات بیشتر به **[اینجا](/docs/developers/api/SimotelWebHooks/ComponentsApi/exten_api)** مراجعه فرمایید).
