

#  Trunk API

برای ایجاد تماس روی ترانک استفاده می‌‌گردد. عملکرد آن مشابه Trunk است، با این تفاوت که از طریق وب‌‌سرویس، نام ترانک و شماره داخلی مقصد را دریافت و تنها تماس را برای آن شماره ایجاد می‌‌کند برای توضیحات بیشتر به **[اینجا](/docs/api/callcenter_api/SimoTelWebHooks/ComponentsApi/trunk_api)** مراجعه فرمایید).
